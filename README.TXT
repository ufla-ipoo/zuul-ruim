Projeto: world-of-zuul-ruim
Autores: Michael Kölling and David J. Barnes

Traduzido por Julio César Alves

Este material é parte do livro

   Objects First with Java - A Practical Introduction using BlueJ
   Sixth edition
   David J. Barnes and Michael Kölling
   Pearson Education, 2016

Este projeto é um framework simples de um jogo do tipo adventure. Nesta
versão há alguns locais e o jogador consegue caminhar entre estes locais.
E isso é tudo.

Para iniciar a aplicação, crie um objeto da classe "Jogo" e chame o método
"jogar".

Esta versão do jogo tem um design de classes muito ruim. Ela NÃO deveria
ser usada como base para estender o projeto, sem antes corrigir os problemas
de design. Esta versão serve como exemplo para discutir designs bons e ruins
(capítulo 8 do livro).

O Capítulo 8 do livro (Cap. 7 na 4ª edição em português), tem uma descrição
detalhada dos problemas deste projeto, e como corrigi-los.

O projeto 'world-of-zuul-melhor' tem uma versão deste projeto com uma
estrutura de classes melhor projetada. Ele inclui as correções discutidas
no livro.
